DOM - Document Object Model

Subheading

Call to action
Preview Area
DOM Controls
Clear

Clears the preview area.

Create Element

Appends a new image HTML tag <img src="..."> into the preview area.

Set Style

Sets a random background color on the body HTML tag.

Set Attribute

Sets/toggles the style and class attributes on all image tags.

Remove Element

Removes the last image tag.

Remove All Elements

Removes all image tags.

Add Event Listener

Adds a mouseover event listener to all image tags. Try to hover the mouse over a preview image and an alert will appear.

Remove Event Listener

Removes mouseover event listeners from all image tags.
